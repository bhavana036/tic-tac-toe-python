Project Title: Tic-Tac-Toe Game in Python

Project Description
This is a console-based Tic-Tac-Toe game built using Python. Two players take turns to place their symbols (X and O) on a 3x3 grid. The game automatically detects wins and draws, providing a simple and interactive way to play Tic-Tac-Toe in the terminal.

Features of the Game

•	Two-player gameplay (Player X and Player O).
•	Detects wins across rows, columns, and diagonals.
•	Detects draws when all spaces are filled.
•	Validates player input to prevent invalid moves.
•	Interactive console-based interface with clear board display.

How to Run
1.	Make sure you have Python 3.x installed on your system.
2.	Open terminal/command prompt and navigate to the folder containing	.
3.	Run the game:
           python tictactoe.py
4.	Follow the on-screen instructions to play the game.

Rules of the Game
1.	The board has 9 positions numbered 1-9:
1	|	2	|	3

4	|	5	|	6
7	|	8	|	9
2.	Players take turns entering a number (1-9) to place their symbol.
3.	The first player to get three of their symbols in a row, column, or diagonal wins.
4.	If all positions are filled and no player has won, the game ends in a draw.
5.	Invalid moves (choosing a filled position or a number outside 1-9) are rejected.
